# Valentine's Day Greeting Card

A romantic, interactive digital greeting card built with React and Framer Motion.

## Features

- **Interactive Envelope**: A 3D-style envelope that opens with a wax seal animation.
- **Story Flow**: A step-by-step guided experience (Envelope -> Letter -> Surprise Gifts -> Music -> Photos).
- **Surprise Gifts**: Three interactive gift cards that reveal your cherished memories (compact size).
- **Music Player**: A dedicated "Song for my Valentine" card featuring "Canon in D".
- **Photo Gallery**: A beautiful, compact gallery of your shared moments.

## Setup & Usage

### Prerequisites
- Node.js 18+
- NPM

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Great Vibes, Playfair Display
